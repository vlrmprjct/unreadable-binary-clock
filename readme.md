# UnBin | Unreadable Binary Clock v1.0.0b

Shows the time in a binary way.
But unreadable for humans.

## PCB
![](doc/3d_pcb.jpg)

## Example
`00:10:01 ===   1001 ===          0011 11101001`

`19:50:50 === 195050 === 0010 11111001 11101010`

`23:59:59 === 235959 === 0011 10011001 10110111`
